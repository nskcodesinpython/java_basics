Exception handling - process of handling exception in Java
                   - Checked Exception - compile time - need to inherit Throwable class 
                   - Unchecked Exception - runtime - need to inherit RuntimeException class 
                   - Error - unrecoverable

5 key words:
  - try
  - catch
  - finally
  - throw
  - throws
